# 📅 Projeto App Agendamento de Consultas — Gestão de Riscos e Comunicação

Este repositório contém os artefatos práticos desenvolvidos para a disciplina de **Gestão de Projetos com Inteligência Artificial Generativa** na pós-graduação. O objetivo é aplicar conceitos avançados de identificação, análise, resposta a riscos e comunicação estratégica em um cenário real de crise controlada em um projeto de software intermediário.

## 📋 Cenário do Projeto
O projeto consiste no desenvolvimento de um aplicativo móvel para **Agendamento de Consultas Médicas**, englobando as funcionalidades de cadastro de usuários, gestão de agenda médica, agendamento de consultas, notificações e integração crítica com um sistema externo de prontuário eletrônico.

### ⚠️ Desafios Atuais Identificados (Fase Intermediária)
* **Instabilidade Crítica na Integração:** Mudanças recentes e falta de documentação clara na API do sistema externo de prontuário.
* **Corrupção de Escopo (*Scope Creep*):** Solicitações frequentes de stakeholders para novas regras de negócio e validações no fluxo core de agendamento.
* **Sobrecarga e Gargalo Operacional:** Equipe enxuta (4 Desenvolvedores e 1 Tester) com sobrecarga de trabalho e atrasos nas estimativas iniciais.

## 🎯 Objetivos da Atividade
1. **Mapear proativamente as ameaças** que comprometem a Linha de Base do projeto (Escopo, Tempo e Qualidade).
2. **Avaliar criticamente o impacto** socio-técnico e operacional das falhas de integração e sobrecarga da equipe.
3. **Desenvolver planos de contingência e mitigação** eficientes, respeitando o limite operacional da equipe de engenharia.
4. **Estruturar uma comunicação transparente e executiva** para stakeholders de alto nível, mitigando ruídos e gerenciando expectativas.
